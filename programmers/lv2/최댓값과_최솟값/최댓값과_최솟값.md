# 최댓값과 최솟값

## 문제 설명

문자열 s에는 공백으로 구분된 숫자들이 저장되어 있습니다. str에 나타나는 숫자 중 최소값과 최대값을 찾아 이를 "(최소값) (최대값)"형태의 문자열을 반환하는 함수, solution을 완성하세요.
예를들어 s가 "1 2 3 4"라면 "1 4"를 리턴하고, "-1 -2 -3 -4"라면 "-4 -1"을 리턴하면 됩니다.

## 제한 조건

s에는 둘 이상의 정수가 공백으로 구분되어 있습니다.

## 입출력 예

|       s       | return  |
| :-----------: | :-----: |
|   "1 2 3 4"   |  "1 4"  |
| "-1 -2 -3 -4" | "-4 -1" |
|    "-1 -1"    | "-1 -1" |

## 문제 풀이

1. 문자열을 숫자로 만들기
2. 숫자끼리 비교하기
3. 최솟값과 최댓값 공백으로 구분하여 출력하기
   이 세가지 단계를 거치면 된다고 생각하였다.

우선 공백으로 구분된 숫자들이 문자열 형태로 저장되어 있기 때문에 문자열을 구분자를 기준으로 나누어 배열에 저장할 수 있는 `split()` 메소드를 사용하여 공백을 기준으로 나누었고 단지 `split(" ")`으로 한다면 공백을 기준으로 문자열의 형태로 저장이 되므로 숫자로 저장하기 위하여 `map`을 사용하여 배열의 각 요소들을 십진법 숫자 형태로 바꿔주었다.

두번째로 숫자끼리 최대 최소를 비교해야하는데 `Math.min()`과 `Math.max()`를 사용하여 최대 최소를 쉽게 구할 수 있기에 이를 이용하였다. 단, 여기에서 숫자로 저장된 배열안의 모든 요소를 확인하여 최대 최소를 구해야하므로 spread 연산자를 사용하였다.

최종적으로 답을 출력할 때 문자열끼리 더하면 공백으로 구분된 출력값을 구할 수 있으므로 단순히 min과 " "과 max를 더하여 출력 결과로 나타내었다.

## 새롭게 배운 점

```
function solution(s) {
    const arr = s.split(' ');

    return Math.min(...arr)+' '+Math.max(...arr);
}
```

추후 더 나은 코드로 개선하기 위해 찾아본 결과 위 코드는 더 간단히 구현할 수 있었다. 또한, `Math.min()`과 `Math.max()`는 주로 숫자를 처리할 때 사용되는 함수로 문자열을 인수로 받을 수 있으며 문자열을 인자로 받았을 때 문자열을 숫자로 변환하여 계산을 시도한다는 점을 새롭게 배웠다.
